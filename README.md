# Scripts Bash - Repositório de Estudos

Este repositório contém exemplos e exercícios em Bash para estudo e prática de shell scripting.

## Como usar

1. Abra um terminal no diretório do repositório.
2. Verifique se o script tem permissão de execução:

   ```bash
   chmod +x caminho/do-script.sh
   ```

3. Execute o script:

   ```bash
   ./caminho/do-script.sh
   ```

4. Para inspecionar o conteúdo de um script, use um editor de texto ou comandos como `cat`, `less` ou `nano`.

## Organização sugerida

- Leia o código do script antes de executá-lo.
- Use `bash script.sh` ou `./script.sh` para rodar os arquivos.
- Adapte e estude cada script para entender a lógica e os comandos utilizados.

## Observações

- Este repositório é voltado para aprendizado e prática de Bash.
- Use os scripts como base para desenvolver suas próprias soluções em shell.
